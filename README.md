#### Step 2 — Clone Repository

```bash
git clone https://github.com/<your-username>/react-dynamic-design.git

cd react-dynamic-design
```

Normally if we want monitor any repository using git first you need to initialize git in that repository using "git init" command.
but here we are first creating repository in github and then going to clone that repository using git so git will automatically get initialize in the local folder after cloning.
